# 欢迎来到我的博客

这是我的第一篇博客文章。我使用 Hugo 创建了这个博客，并选择了 PaperMod 主题。

## 为什么选择 Hugo？

Hugo 是一个用 Go 语言编写的静态网站生成器，它具有以下优点：

1. 速度快
2. 易于使用
3. 主题丰富
4. 支持 Markdown
